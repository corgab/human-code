- ordinare

    - chiamo la pizzeria 
    - ordino la pizza prosciutto e funghi
    - prendo la pizza
    - la mangio

- post mangiato
    
    - se finisco la pizza
    - butto il cartone 

- altrimenti prendo la pizza rimasta
    
    - la poso dentro al frigo
    - butto il cartone

- vado a letto
